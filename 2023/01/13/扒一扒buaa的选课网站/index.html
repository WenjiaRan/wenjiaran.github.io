扒一扒buaa的选课网站

发表于 2023-01-13 更新于 2023-01-16 分类于 buaa 阅读次数：
本文字数： 1.2k 阅读时长 ≈ 2 分钟

又到了半年一度的选课的时间, 正好在放假, 抽空扒一扒buaa教务网站的前端(主要是后端咱也看不到啊:(

麻利输完账号密码，然后进入教务官网, 得再输一次, 登录提交!

抓一个包, 看看提交了哪些东西

好家伙，密码都不加密呢，直接发一个post请求把账号密码等数据发到服务器

一顿操作，终于进入选课页面

然后摁开F12看看html源码，html里值得关注的有几个函数：

//切换角色
function qhjsLogin(jsdm){//qhjs：切换角色；jsdm：角色代码
	if($("#dljs").val() == jsdm){
		return;
	}else{
		$("#jsdm").val(jsdm);
		document.queryform.action = "/ieas2.1/login";//这是登录界面的网址
	    document.queryform.submit();//然后转到登录界面重新登陆来切换身份，角色代码根本没啥用其实
	}
}

好家伙，拼音缩写变量函数啊。为了方便理解，我在上面翻译一下，给了一些注释

对函数一波分析完之后，原来这个函数没啥值得关注的:(

然后还有切换角色、返回主页、修改密码、切换到新版界面等，不谈

然后正式开始选课

首先是体育选课，选课按钮上绑定了一个函数， saveXsxk_qz('2022-2023-2-B2E332060-157', '0');

Xsxk就是学生选课，qz就是权值，体育选课是有一个权重的，0到100

在源代码里找到这个函数

function saveXsxk_qz(rwh,rownum){...}
//rwh是任务号，rownum是序号

我们分析一下上面的rwh的格式，'2022-2023-2-B2E332060-157'

2022-2023-2就是2022-2023年的第2学期，B2E332060是课程代码，B2E33应该是体育，2060不详

此外还有志愿学生选课和学生选课两个函数，都差不多

然后是一个比较重要的函数：function saveXsxk1(rwh);

通过这个函数可以直接选课

然后我们点击一下选一门课试试，可以看到浏览器向https://d.buaa.edu.cn/http-8080/77726476706e69737468656265737421fae0598869327d517f468ca88d1b203b/ieas2.1/xslbxk/saveXsxk发送了一个POST请求

我们分析一下抓包，可以看到，里面携带了rwh，看来我们可以通过更改rwh改变选课

我们刚才选了课，现在退课一试，退课调用函数saveXstk('2022-2023-2-B3J212731-001'); 也发送了一个退课的POST请求
